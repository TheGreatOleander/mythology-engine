# Release Review and Publish Layer

v1.9 introduces a release inspection and publish abstraction layer.

Commands:

    mythology --action review-release
    mythology --action publish-sim

Review-release:
    Inspects artifacts produced by the episode pipeline.

Publish-sim:
    Builds a publish payload and simulates sending it to platforms.

Future providers:
    YouTube
    TikTok
    Instagram Reels
